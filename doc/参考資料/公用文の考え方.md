# 公用文作成の考え方（要約）

## 1. 基本的な考え方
* **目的と対象:** 読み手とのコミュニケーションを重視し、目的に応じて文書を作成する。
* **正確さと分かりやすさ:** 告示等は原則に従って正確に書き、広く一般に向けた広報等は、義務教育で学ぶ範囲の知識で理解できるよう平易に書く。
* **気持ちへの配慮:** 読み手に違和感を与えないよう配慮し、対外的な文書は「です・ます」体を基本として簡潔に敬意を表す。

## 2. 表記の原則
* **漢字:** 「常用漢字表」に基づく。表にない漢字や音訓は仮名書きにするか、別の分かりやすい言葉に言い換える。
* **送り仮名:** 「送り仮名の付け方」に基づく。「手続」「申込み」「話合い」など、読み間違えるおそれのない複合の語の名詞は送り仮名を省く。
* **外来語:** 「外来語の表記」に基づき、日本語として広く使われている表記を用いる。長音は原則として長音符号（ー）を使う。
* **数字:** * 横書きは算用数字を用い、4桁以上の数は3桁ごとにコンマで区切る。
  * 「兆・億・万」の単位や概数、ことわざ等の語を構成する数は漢数字を使う（例：5兆、数十人、三日坊主）。
* **符号:** * 句点は「。」、読点は「、」を用いる（横書きでは「,」も可だが文書内で統一する）。
  * 括弧は「」と（）を基本とし、括弧の中で文が終わる場合は句点（。）を打つ。

## 3. 用語と文体の使い方
* **特有の用語:** 「及び・並びに」「又は・若しくは」は、法令に準ずる文書では正確に使い分けるが、一般向けでは分かりやすく言い換える。
* **専門用語・外来語:** 一般になじみのない専門用語や外来語は、日常語に言い換えるか、意味を明確にする説明を付けて使う。
* **紛らわしい言葉:** 同音異義語の混同を避け、「等」「など」の語はむやみに使わず、回りくどい表現や重複を避ける。
* **文体と構成:** * 一文は短くし（目安として50〜60字程度）、一つの文の論点は一つに絞る。
  * 三つ以上の情報を並べる時は、箇条書きを利用して視覚的に分かりやすくする。
  * 結論は文書の早めに示し、続けて理由や詳細を説明する構成にする。
